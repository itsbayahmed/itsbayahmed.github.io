# Things I Do After Installing Fedora Workstation:


## Updating the system:

> sudo dnf upgrade --refresh -y

## Installing some useful missing apps:

1. ### Gnome Tweaks to add *maximize* and *minimize* buttons to the apps window:

> sudo dnf install gnome-tweaks

2. ### [Adw-gtk3](https://github.com/lassekongo83/adw-gtk3) to make GTK 3 apps look like GTK 4/ Libadwaita apps:

> sudo dnf install adw-gtk3-theme

3. ### [Brave](https://brave.com/) Web Browser:

> curl -fsS https://dl.brave.com/install.sh | sh 

4. ### [Obsidian](https://obsidian.md/): The Best Note taking App

> flatpak install flathub md.obsidian.Obsidian

5. ### Install Geary email client:
> sudo dnf install geary
